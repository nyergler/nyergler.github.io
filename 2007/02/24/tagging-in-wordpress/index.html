Tagging in WordPress

24 February 2007

Dear LazyWeb,

OK, I’m convinced. Heirarchical categories for blogging are a pain in the ass. I’d rather tag things, and like that WordPress automagically add rel=“tag” to my category links. But I hate the check box selector, since I can type faster than I can mouse. Is there a plugin that will give me a single text field in the editor which:

  • auto-completes space-delimited words to existing categories
  • and if they don’t exist, creates them when the post is published?

Eternally grateful,

Nathan

UPDATE Apparently I really am “that lazy”; check out cat2tag which does 99% of what I want (it doesn’t do tab-completion, but I can live with that).
